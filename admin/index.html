Bosch SingleKey ID Anmeldung

1

Klicke auf „Login-URL öffnen" — die Bosch-Anmeldeseite öffnet sich in einem neuen Fenster. Nutze ein privates/Inkognito-Fenster.

2

Melde dich mit deinen Bosch SingleKey ID Zugangsdaten an.

3

Nach dem Login erscheint ein Redirect-Fehler — das ist normal. Öffne Entwicklertools (F12) → Netzwerk, filtere nach pointt, kopiere den code-Parameter (endet auf -1).

4

Code unten einfügen und „Verbinden" klicken.
